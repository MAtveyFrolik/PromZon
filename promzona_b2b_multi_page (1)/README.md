# PromZona B2B — Федеральный реестр промышленности

Данный архив содержит полностью рабочий, автономный мультистраничный проект!

## Особенности архитектуры:
1. **Чистый HTML5 & Vanilla JS & Tailwind CSS** — весь проект собран в виде отдельных HTML файлов.
2. **Нулевой порог запуска** — нет сложных сборщиков, TypeScript, или Vite конфигов для локальной работы.
3. **Локальный роутинг** — переходы между страницами осуществляются по классическим ссылкам (например, `catalog.html`).
4. **Сохранение состояния** — добавление объявлений, диалоги в чате, фильтры синхронизируются через `localStorage` браузера!

## Как запустить:
1. Распакуйте архив.
2. Запустите проект через локальный веб-сервер. Например в VS Code:
   - Установите расширение **Live Server** и нажмите **Go Live**.
   - Либо из консоли в папке проекта:
     ```bash
     npx serve
     ```
     или
     ```bash
     python -m http.server 8000
     ```
3. Откройте адрес сервера в любом браузере и наслаждайтесь!
